# web3-aa

### `@paro-paro/web3-aa`

Custom web3 account abstraction integration using [Safe{CORE} SDK's](https://github.com/safe-global/safe-core-sdk) provided by [Safe Global](https://docs.safe.global/getting-started/readme).

This package is not meant for public consumption.

Safe AA packages:
* [protocol-kit](https://docs.safe.global/safe-core-aa-sdk/protocol-kit)
* [api-kit](https://docs.safe.global/safe-core-aa-sdk/api-kit)
* [relay-kit](https://docs.safe.global/safe-core-aa-sdk/relay-kit)
